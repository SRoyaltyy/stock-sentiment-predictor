# Stock Sentiment Predictor (1-Week & 2-Week Forecasts)

100% free, runs entirely in GitHub Actions — no local install needed.

## How it works
- Daily automatic run via GitHub Actions
- Uses fresh X sentiment (updated by SuperGrok team)
- yfinance for prices + technical indicators
- XGBoost for predictions

## View latest predictions
Check the **Actions** tab → latest workflow run → logs

SuperGrok team will keep the X sentiment updated automatically.
